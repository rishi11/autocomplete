Implementation Details

  • The comes with some predefined list of fruits
  • As user enters 3 or more characters the fruits comes in drop down menu filtered from the list
  • As user enters more characters the list become shorter
  • The list is ordered by POPULARITY of fruit
  • If the fruit is not present and user types fruit name and presses SEARCH button that fruit gets added to list of fruits
